선택자와 속성

저번 페이지에서 보았듯이, 웹페이지에 우리가 CSS를 포함시키기 위해서는 웹브라우저로 하여금 어디부터 어디까지가 CSS인지 헤더에 style태그를 포함시켜 알릴 필요가 있었다.

하지만 이외에도 한가지 방법이 더 존재하는데, 헤드에 style태그를 선언하는 것이 아니라 본문(body)상 적용시키고싶은 코드의 태그 내에 style이라는 속성을 추가시키는 방법이다. 그 적용예시는 아래와 같다.

설명한 CSS의 스타일 속성이 적용된 부분이다.

<c style="color:deeppink">설명한 CSS의 스타일 속성이 적용된 부분이다.</c>

선택자

태그명 {
속성:속성값;
}

위의 첫번째 방식에서 태그명과 중괄호를 선택자(selector)라고 하며 중괄호 내의 내용인 속성:속성값; 을 선언(declaration 또는 효과)이라고 한다.

속성

<태그명 style="속성:속성값">내용</태그명>

의 두번째 방식에서는 스타일 태그를 해당하는 태그내에 직접 위치시켜 선택자 없이 직접 작용한다.



번외:밑줄 제거

헤더에 포함된 선택자로 지정된 태그에 대한 선언(중괄호 내 내용)에

text-decoration: none;

을 포함시킨다. 이런 상황에서 특정 라인에 대해서만 밑줄을 포함시키고 싶다면 해당 태그에 직접 style태그를 포함시키고 style="text-decoration: underline" 같이 적어주도록 한다.



<head> a { text-decoration: none; } </head>
