BOXMODEL

HTML 요소가 브라우저 엔진에서 모델링하고 그 HTML 요소의 치수 CSS 속성에서 방법을 유도하는 방법을 말한다. 그것은 HTML 웹 페이지의 구성에 대한 기본적인 개념이다.

이 박스모델에서 각 html태그 요소들의 위치와 부피감을 알기위해 stlye의 border- 를 활용할수 있다. 이를 활용하여 확인했을떄 화면 전체의 공간을 사용하는 element가 있고 (ex.h1,h2..) 반대로 자기 자신의 크기만큼의 공간만 사용하는 element가 있다. (ex.href..)
이 둘을 각각 Block level element 와 in-line element라고 한다.

이 때 block line element의 style에 display:inline; 의 선언을 추가해주면 inline element로 전환할 수 있고 반대도 가능하다.

+중복제거 TIP1)
선택자를 지정할 때 콤마를 사용하여 한번에 다수의 element를 선택할 수 있다.

+중복제거 TIP2)
하단에 나올것처럼 지정한 선택자의 border attribute 3개를 하나의 줄로 묶어서 선언하여 중복을 제거할 수 있다.

두개의 중복제거 코드의 모습은 아래와 같다.

<head> <style> h1 { border-width: 2px; border-color: red; border-style: solid; } a { border-width: 2px; border-color: red; border-style: solid; } </style> </head> <head> <style> h1,a { border: 2px solid red; } </style> </head>

+TIP padding 속성에 px값을 주면 element와 border 테두리와의 간격을 발생시킬 수 있다.

+TIP margin 속성에 px값을 주어 elements간의 간격을 발생시킬 수 있다.

+TIP CSS BOX MODEL의 4요소는 가장 안쪽부터 content, padding, border, margin까지 네개로 구성된다.
